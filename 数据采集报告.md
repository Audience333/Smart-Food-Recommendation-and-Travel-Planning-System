## 使用方法

### 方式一：直接打开
1. 双击打开 `map.html` 文件
2. 使用鼠标滚轮缩放地图
3. 拖拽地图移动视图
4. 点击标记查看详细信息
5. 使用左上角面板切换图层
6. 使用右上角面板快速定位商圈
7. 使用顶部搜索框搜索景点或美食

### 方式二：重新生成数据
1. 运行 `python chengdu_map.py` 采集最新数据
2. 运行 `python update_food_data.py` 添加美团数据
3. 运行 `python generate_map.py` 生成地图
4. 打开 `map.html` 查看结果

## 技术栈

- Python 3
- 高德地图 JavaScript API 2.0
- 高德地图 Loca 2.0（热力图）
- JSON 数据格式
- CSS3 渐变和动画

## 截图说明

请在浏览器中打开 `map.html`，然后使用以下方法截图：

### Windows
- 使用 `Win + Shift + S` 进行区域截图
- 使用 `Print Screen` 键全屏截图

### Mac
- 使用 `Cmd + Shift + 4` 进行区域截图
- 使用 `Cmd + Shift + 3` 全屏截图

### 推荐截图
1. **全览图** - 显示所有景点和美食标记
2. **景点图层** - 仅显示景点标记
3. **美食图层** - 仅显示美食标记
4. **热力图** - 显示美食分布密度
5. **商圈聚类** - 显示各商圈的聚类效果
6. **信息窗口** - 点击标记后的信息展示（含美团评分）
7. **搜索功能** - 搜索特定景点或美食

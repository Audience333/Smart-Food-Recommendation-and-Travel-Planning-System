# 地图截图目录

请在浏览器中打开 `map.html`，然后使用以下方法截图：

## 推荐截图

1. **全览图** - 显示所有景点和美食标记
2. **景点图层** - 仅显示景点标记
3. **美食图层** - 仅显示美食标记
4. **热力图** - 显示美食分布密度
5. **商圈聚类** - 显示各商圈的聚类效果
6. **信息窗口** - 点击标记后的信息展示

## 截图方法

### Windows
- 使用 `Win + Shift + S` 进行区域截图
- 使用 `Print Screen` 键全屏截图

### Mac
- 使用 `Cmd + Shift + 4` 进行区域截图
- 使用 `Cmd + Shift + 3` 全屏截图

### 浏览器
- 使用浏览器开发者工具的截图功能
- Chrome: `F12` -> `Ctrl + Shift + P` -> `Capture screenshot`

## 保存格式

建议保存为 PNG 格式，文件名示例：
- `01_overview.png` - 全览图
- `02_scenic_only.png` - 仅景点
- `03_food_only.png` - 仅美食
- `04_heatmap.png` - 热力图
- `05_cluster.png` - 聚类效果
- `06_info_window.png` - 信息窗口

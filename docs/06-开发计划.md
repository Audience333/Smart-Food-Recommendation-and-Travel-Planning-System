## 四、技术风险评估

| 风险项 | 风险等级 | 应对策略 |
|--------|----------|----------|
| 高德地图 API 限制 | 中 | 申请企业版或使用开源方案 |
| 推荐算法效果不佳 | 中 | 先用简单算法，逐步优化 |
| 数据采集困难 | 高 | 使用公开数据 + 人工采集 |
| AI 大模型成本 | 中 | 控制调用频率，缓存结果 |
| 性能瓶颈 | 低 | 使用缓存，优化查询 |

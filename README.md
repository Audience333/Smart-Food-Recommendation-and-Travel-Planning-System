# 智慧美食推荐与漫游规划系统

> 高德地图 Web 可视化 + Python 数据管道

## 项目简介

以菏泽市 **288** 条真实美食门店和 **110** 个景点数据为基础，通过 Python 数据管道对接高德地图 API 获取真实 POI 数据（含坐标、评分、地址、营业时间、门店照片），前端提供完整的地图交互：搜索筛选、驾车/步行路线规划、收藏管理、用户画像分析、一日游智能推荐、排行榜等功能。

## 技术栈

| 层次 | 技术 | 用途 |
|------|------|------|
| 数据管道 | Python 3 + urllib | 高德API数据采集/地址补全/照片获取/道路计算 |
| Web 前端 | Vanilla JS + 高德地图 JS API 2.0 | 地图可视化、人机交互 |
| 数据存储 | TXT (管道分隔) + JSON + localStorage | 源数据 + 前端数据 + 用户状态 |

```
│   ├── gen_web_json.py           # TXT → JSON 转换
│   ├── recalc_roads.py           # 道路连接重算 (+ 驾车API)
│   └── fetch_amap_data.py        # 原始高德 POI 采集脚本
│
└── web/                          # Web 前端
    ├── index.html                # 主页面 (11个功能面板)

        ├── spot.json             # 景点 (110条)

```

## 数据管道

```
高德地图 API (restapi.amap.com)
    │
    ├── expand_data.py    → 调用 place/text 搜索POI → food.txt / spot.txt
    │                       + 12维度标签自动生成
    │
    ├── fill_addresses.py → 调用 geocode/regeo 逆地理编码 → 补全地址
    │
    ├── fetch_photos.py   → 调用 place/detail 获取门店照片 → food.txt / spot.txt
    │
    ├── recalc_roads.py   → 调用 direction/driving 驾车路径 → road.txt
    │                       + Haversine 降级策略
    │
    └── gen_web_json.py   → TXT → JSON 转换 → web/data/*.json
```

## Web 前端功能

### 图层控制
- 美食标记 / 景点标记 按钮式开关
- 仅显示营业中 筛选
- 美食分类标签筛选（点击自动缩放地图到对应区域）

### 搜索
- 实时搜索美食/景点名称、地址、标签
- 搜索结果列表，点击定位到地图标记
- 支持全部/美食/景点类型切换

### 数据统计
- 美食数量 / 景点数量 实时显示

### 路线规划
- 通行方式选择：驾车 / 步行
- 起点终点搜索选择（输入框实时过滤）
- 途经点添加/删除/排序（最多5个）
- 排序策略：时间优先 / 距离优先 / 过路费少
- 调用高德驾车/步行API获取真实路径距离/时间/过路费
- 路线详情：分段距离/时间/过路费展示

### 美食分类
- 8 大分类标签筛选（汤类/面食/小吃/正餐/烧烤/甜品/饮品/凉菜）
- 多选筛选，实时更新地图标记

### 用户画像
- 口味偏好标签云（自动从收藏分析）
- 价格区间分布条形图
- 偏好品类占比
- 编辑模式：+/- 调整权重 / 从已存在标签中选择添加 / 删除
- 保存 / 取消 / 恢复默认

### 一日游推荐
- 智能算法：根据收藏画像匹配偏好品类 → 筛选高分美食 → 找最近景点
- 生成3个候选方案（含评分/距离/预计时间）
- "换一批"随机刷新
- "采纳"一键填入路线规划并自动执行

### 排行榜
- 人气榜（评分降序）、性价比榜（评分/价格对数比值降序）、收藏榜
- 前3名奖牌展示
- 正序/倒序切换
- 点击排名项跳转地图定位

### 收藏夹
- 收藏/取消收藏（详情浮窗顶部按钮 + 标题栏收藏面板）
- localStorage 持久化
- 收藏标记金色边框

### 操作历史
- 标题栏撤销 / 重做按钮
- 历史记录下拉面板（筛选/搜索/收藏/路线操作）
- 点击历史项跳转到对应状态

### 详情浮窗
- 评分星级 / 价格 / 分类 / 地址 / 营业时间
- 高德真实门店照片（最多3张）
- 12 维度标签展示
- 点击标记可切换显示/关闭浮窗
- 点击地图空白处关闭浮窗

## 快速开始

### 前端运行

```bash
cd HezeFoodSystem/web
python -m http.server 8080
# 浏览器访问 http://localhost:8080
```

### 数据管道（更新数据时使用）

```bash
cd HezeFoodSystem

# 获取最新高德 POI 数据
python tools/expand_data.py

# 补全缺失地址
python tools/fill_addresses.py

# 获取门店照片
python tools/fetch_photos.py

# 重新计算道路连接
python tools/recalc_roads.py

# 生成前端 JSON
python tools/gen_web_json.py
```
